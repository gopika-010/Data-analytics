##  Overview
This analysis focuses on identifying the **highest scores across different subjects**.  
A key feature of this dashboard is the use of **interactive slicers**, which allow users to dynamically filter and view subject-wise performance.

##  Tool Used
- **Power BI**

##  Dataset
- **Source:** Kaggle  
- **File Name:** `kaggle_dataset.csv`  
- Contains subject-wise student score data

##  Visualizations Included
- Bar chart displaying the **highest score for each subject**
- Slicers to filter subjects and analyze data dynamically

##  Key Insights
- Enables easy comparison of highest scores across subjects
- Interactive slicers improve data exploration and flexibility
- Makes academic performance analysis clearer and more effective
