 Overview :
     This dashboard shows the highest marks scored in each subject.
     It helps identify top performance across subjects using clear visual comparisons.
 Tool Used :
     Power BI
 Dataset :
     File: Marks.csv
     Contains student-wise marks across multiple subjects.
 Visualizations Included :
     Bar chart showing highest score per subject
     Easy comparison across subjects
 Key Insights :
    Highlights top-performing subjects
    Helps understand subject-wise score distribution
    Useful for academic performance analysis
